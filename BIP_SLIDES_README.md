# BIP Slide HTML Replicas

This directory contains 13 HTML replicas of BIP presentation slides. Each file is a self-contained HTML document using Tailwind CSS.

## Viewing the Slides

### Option 1: Using GitHub Pages (Recommended)
If GitHub Pages is enabled for this repository, you can view the slides at:
- `https://[username].github.io/[repository-name]/bip-slide-1.html`
- Replace `[username]` and `[repository-name]` with your GitHub details

## Technical Details

- **Framework**: Tailwind CSS (via CDN)
- **Font**: Inter (Google Fonts)
- **Features**:
  - Responsive layouts
  - Red dotted underlines for emphasis
  - White dotted underlines on dark backgrounds
  - No graphics, logos, or page numbers (text only)
  - Self-contained (no external dependencies except CDN)
